POSIX file

  1. 与 'file' 类等效的伪类。
  2. 从来没有一个对象的类是 'POSIX file '；
  3. 评估 POSIX文件说明符的结果是一个 “文件” 对象
  4. “文件” 和 “posix文件” 对象之间的区别在于它们如何解释名称说明符
  5. a POSIX file object interprets "name" as a POSIX path, while a file object interprets it as an HFS path.
  6. For related information, see alias and file. For a description of the format for a POSIX path, see Aliases and Files.

Examples
