callbacks简介

这里的callback可以认为是cglib用于生成字节码的实现手段，cglib一共实现了6种callback，用于对代理类目标进行不同手段的代理

分别为：

  • FixedValue
  • InvocationHandler
  • LazyLoader
  • MethodInterceptor
  • Dispatcher
  • NoOp
  • ProxyRefDispatcher

Dispatcher

实现Dispatcher接口，要求实现loadObject方法，返回期望的代理类。

值的一提的是，loadobject方法在每次调用被拦截方法的时候都会被调用一次

手动返回指定的被代理类的，每次调用都会返回一个新的
