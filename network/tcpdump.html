计算机网络

命令格式

tcpdump [选项] [表达式]

选项

指定监听的网络接口

tcpdump -i eth0

不进行主机名解析，直接显示IP地址。

tcpdump -n

不进行主机名和端口号解析。

tcpdump -nn

-v, -vv, -vvv：增加详细输出的级别。

tcpdump -vv

-c count：捕获指定数量的数据包后停止。

tcpdump -c 10

-w file：将捕获的数据包写入文件。

tcpdump -w capture.pcap

-r file：从文件中读取数据包进行分析。

tcpdump -r capture.pcap

-s snaplen：指定捕获的每个数据包的最大长度。

tcpdump -s 0


-s 0 表示捕获整个数据包。

-A：以ASCII格式显示数据包的内容。

tcpdump -A

-X：以十六进制和ASCII格式显示数据包的内容。

tcpdump -X

表达式

主机：捕获特定主机的流量。

tcpdump host 192.168.1.1

网络：捕获特定网络的流量。

tcpdump net 192.168.1.0/24

端口：捕获特定端口的流量。

tcpdump port 80

协议：捕获特定协议的流量（如TCP、UDP、ICMP）。

tcpdump tcp
tcpdump udp
tcpdump icmp

源地址和目标地址：捕获特定源或目标地址的流量。

tcpdump src 192.168.1.1
tcpdump dst 192.168.1.1

布尔表达式

且

tcpdump src 192.168.1.1 and dst port 80

或

tcpdump src 192.168.1.1 or src 10.0.0.1

非

tcpdump not port 22
